# Audio Assets for PartyGame

This directory contains audio files used by the TV presentation layer.

## Required Audio Files

### Background Music (Loops)
- `intro_loop.mp3` - Upbeat intro music (loops)
- `category_loop.mp3` - Calm music during category selection (loops)
- `question_loop.mp3` - Building tension during question display (loops)
- `tension_loop.mp3` - High tension during answering phase (loops)
- `reveal_sting.mp3` - One-shot sound for answer reveal
- `scoreboard_loop.mp3` - Celebratory music for scoreboard (loops)
- `victory_sting.mp3` - Victory fanfare for game end

### Sound Effects
- `tick.mp3` - Countdown tick (last 10 seconds)
- `buzzer.mp3` - Time's up buzzer
- `booster.mp3` - Booster activation sound
- `correct.mp3` - Correct answer sound
- `wrong.mp3` - Wrong answer sound
- `splash.mp3` - Quiz start splash sound

## Replacing Assets

1. Replace any file with a new MP3 of the same name
2. Keep files under 500KB for optimal loading
3. Use consistent volume levels across files
4. Loops should seamlessly repeat

## Volume Guidelines

- Background music: Normalize to -12dB
- Sound effects: Normalize to -6dB
- Ensure no clipping

## Licensing

When replacing with custom audio:
- Ensure you have proper licensing
- Royalty-free or original compositions recommended
- Credit artists as appropriate
